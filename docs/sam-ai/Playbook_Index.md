# SAM v4.4.1 Master Playbook Index (Annotated)

This index provides an overview of all files, grouped by category, with short summaries.

---

## Core Product Knowledge (/core)
- **SAM_Product_OnePager.md** → Quick overview of SAM, the problem it solves, and pricing tiers. Use for client-facing intros.  
- **SAM_Messaging_Playbook.md** → Pain points mapped to SAM’s solutions with approved phrasing. Foundation for outreach copy.  
- **SAM_Case_Studies.md** → Success stories (SaaS, Consulting, Finance, etc.) with proof metrics. Great for pitches.  
- **SAM_Objection_Handling.md** → Objection library with human-sounding responses. Use during demos and sales convos.  
- **ROI_Calculator_Template.xlsx** → Spreadsheet to compare SDR costs vs SAM.  

## Conversational Design (/conversational-design)
- **Conversational_Principles.md** → Tone, microbursts, skip/resume rules, natural conversation patterns.  
- **Conversation_Modes.md** → Modes: Onboarding, Product QA, Campaign, Repair (with context switching).  
- **Onboarding_Flow.md** → 7-stage onboarding flow with lead questions, follow-ups, and transitions.  
- **Full_Onboarding_Flow.md** → Expanded onboarding script with variants, clarifications, and microbursts.  
- **QA_Library.md** → Example MUST_HAVE / NICE_TO_HAVE / PROBE questions with sample responses.  
- **Error_Handling.md** → Strategies for contradiction repair, frustration detection, and recovery scripts.  
- **Example_Conversations.md** → Example dialogue flows: product-first, onboarding-first, repair mode, etc.  
- **Product_QA.md** → Answers about SAM’s features, pricing, integrations, agents, compliance, verticals.  
- **Industry_Bursts.md** → Quick industry-specific dialogue openers (SaaS, Finance, Consulting, Pharma).  

## Strategy (/strategy)
- **ICP_Deep_Dive.md** → Framework and question packs to define and refine Ideal Customer Profiles.  
- **Competitive_Intelligence.md** → Tools to capture competitor analysis and objection handling insights.  
- **UVP_Value_Proposition_Builder.md** → Template to build compelling UVPs with prompts and examples.  
- **Campaign_History_Strategy.md** → Captures past campaign learnings, successful templates, and failures.  

## Agents (/agents)
- **Sub_Agent_Capabilities.md** → Profiles of SAM’s 14 sub-agents with customer-facing language.  
- **Inbound_SDR_Agent_Guide.md** → Inbound SDR workflows with dialogue examples for demos & info requests.  
- **Agent_Orchestration_After_Onboarding.md** → How SAM transitions from onboarding → orchestration.  

## Industry Verticals (/verticals)
- 20+ templates (SaaS, Consulting, Recruiting, Finance, Pharma, Legal, Manufacturing, Coaching, AI companies, SMEs, etc.).  
Each template contains: ICP patterns, compliance notes, microbursts, KPIs.  

## Compliance & Admin (/compliance)
- **Finance_Guide.md** → Compliance guidelines for financial services.  
- **HIPAA_Guide.md** → Privacy and compliance for healthcare data.  
- **FDA_Promo_Messaging_Guide.md** → Rules for compliant pharma messaging.  
- **Legal_Guide.md** → Legal and state-specific compliance notes.  
- **HITL_Compliance_Workflows.md** → Human-in-the-loop approval checkpoints.  
- **PHI_Data_Handling_Policy.md** → Protected Health Information handling rules.  
- **PreApproved_Disclaimers_Library.md** → Standard disclaimers for regulated industries.  
- **State_Regulations_Reference.md** → Overview of US state-specific legal/financial rules.  
- **ADMIN_DATABASE_FOUNDATION.md** → Multi-tenant DB + RLS structure, admin controls, audit trails.  

## Prospect Workflows (/prospect-workflows)
- **prospect-workflow-specifications.md** → Detailed ICP → list building → enrichment → approval → outreach.  
- **conversation-workflow-design.md** → Conversation-driven workflow for moving from ICP → research → extended research.  
- **DATA_SCRAPING_ENRICHMENT_STRATEGY.md** → Defines scraping sources, enrichment logic, and quality gates.  
- **knowledge-classification-proposal.md** → Taxonomy proposal for labeling data in the KB.  
- **KNOWLEDGE_CLASSIFICATION_DEPLOYMENT_GUIDE.md** → Deployment playbook for classification in Supabase RAG.
